# Verdaccio npm registry

A private/local npm registry server

## Requirements

- Docker version 17.x

- Docker compose 1.11.x

## How to play

- Clone that repo

- Go to folder repo cloned

- Run that command for create and start the server with docker:

```

~ docker compose up -d 

```
